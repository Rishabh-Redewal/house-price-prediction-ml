# House Price Prediction ML

A Machine Learning project built using Scikit-learn to predict house prices through data preprocessing and ML pipelines.

## Features
- Data cleaning and preprocessing
- House price prediction using Machine Learning
- Data transformation pipeline
- CSV input and prediction output support

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Project Files
- `main.py` → Main source code
- `housing.csv` → Dataset used
- `pipeline.pkl` → Preprocessing pipeline
- `input.csv` → Input sample data
- `output.csv` → Prediction output
